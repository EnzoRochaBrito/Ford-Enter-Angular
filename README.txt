
FordEnterAngular

Este projeto foi gerado usando o Angular CLI versão 19.2.7.

---

Para instalar as dependências necessárias, abra o seu terminal e digite:

npm install

---

Servidor de Desenvolvimento

Para iniciar o servidor de desenvolvimento local, execute:

ng serve

Após o servidor estar em execução, abra o navegador e acesse:

http://localhost:4200/

O aplicativo será recarregado automaticamente sempre que você modificar algum arquivo fonte.

---

Executar a API

Para rodar a API, use o seguinte comando:

npm run api
